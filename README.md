# svvm
The SVVM is reference implementation of SVasm, smallest stack base vm for education.

## Features
- Simple and smallest struct.
- Understand easily.
- Able use as library and command.

## How to Install
### Command
``` bash
cargo install svvm-cli
```

### Library
``` bash
cargo add svvm_core
```
